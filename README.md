🚀 The MissionThe week is starting, and the "bad vibes" are raining down.
Navigate through 5 distinct levels—from the "Monday Blues" to the "Friday Frenzy"—collecting positive energy to fill your progress bar and reach the weekend.
🕹️ Features
•Dynamic Difficulty: Game speed and spawn rates increase as you progress from Monday to Friday.
•Vibe Collection System:•Good Vibes: Catch ☕ (Coffee), 🍕 (Pizza), 🎮 (Gaming), and ✨ (Zen) to boost your progress.
•Bad Vibes: Avoid 📧 (Emails), 📑 (Reports), and 💣 (Deadlines) or lose lives!
•Smooth Mechanics: Features screen-shake effects on impact
🛠️ Tech Stack
•Android/Kotlin: Native host application using AppCompat and WebView.
•Frontend: HTML5, CSS3 (Flexbox/Grid), and Vanilla JavaScript.
•Graphics: HTML5 Canvas API for 60FPS rendering.
•UI/UX: Google Fonts (Bangers & Fredoka) for a bold, arcade-style aesthetic.
📱 How to Run
1.Clone this repository.
2.Open the project in Android Studio.
3.Ensure index.html and your assets (like gf_photo.png) are located in app/src/main/assets/.
4.Build and run on an emulator or physical device (Android 7.0+ recommended).
